# Code of Conduct

## Our standards
We are committed to providing a respectful, inclusive environment.

Expected behavior:
- Be respectful and constructive
- Assume good intent
- Give actionable feedback

Unacceptable behavior:
- Harassment or discrimination
- Personal attacks
- Publishing private information

## Scope
This code of conduct applies in project spaces (issues, pull requests, discussions).

## Enforcement
If a problem arises, pause the conversation and ask for help from maintainers.
